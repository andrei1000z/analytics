Analytics Tracker — Demo

Ăsta e un demo local pentru tracker. Snippet-ul încarcă tracker-ul de la ../dist/tracker.js, ratează request-ul de ingest (nu rulează un server local), dar îl poți inspecta în Network tab → /e request cu body application/octet-stream (12B IV + ciphertext).

Verificare manuală

  1. pnpm --filter @analytics/tracker build
  2. Servește acest folder cu npx serve packages/tracker/demo
  3. Deschide DevTools → Network → Filter „/e"
  4. Click pe butoane mai jos. Fiecare emite un beacon criptat AES-GCM.

Snippet

<script
  src="https://cdn.example.eu/t.js#<base64url-key>"
  data-site="<site-id-pbkdf2-hash>"
  defer
></script>
